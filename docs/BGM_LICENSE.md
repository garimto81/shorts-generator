## 크레딧 표기 방법

### 방법 1: 영상 설명 (권장)

```
Music Credits:
- [Artist] - [Song Title] [NCS Release]
Music provided by NoCopyrightSounds
https://ncs.io
```

**예시:**
```
Music Credits:
- Elektronomia - Sky High [NCS Release]
Music provided by NoCopyrightSounds
https://ncs.io
```

### 방법 2: 영상 내 자막/엔드카드

영상 마지막 2-3초에 크레딧 표시

### 방법 3: 핀 댓글

YouTube 영상의 첫 번째 댓글로 크레딧 고정
