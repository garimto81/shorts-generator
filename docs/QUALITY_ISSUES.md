# Shorts Generator 콘텐츠 퀄리티 이슈

> 관련 이슈: [#9 콘텐츠 퀄리티 종합 개선](https://github.com/garimto81/shorts-generator/issues/9)

## 현재 한계점

### 1. 영상 효과

| 항목 | 현재 상태 | 문제점 |
|------|----------|--------|
| Ken Burns | 8가지 패턴 (zoom + pan 조합) | 해결됨 |
| 전환 효과 | xfade 10종 | 실제 활용 제한적 |
| 움직임 | 다양한 방향 + 줌 조합 | 해결됨 |

**완료:** Ken Burns 패턴 다양화
- 8가지 패턴: zoomInCenter, zoomOutCenter, panLeftTopToRightBottom 등
- 3가지 모드: classic (기존), sequential (순환), random (무작위)
- CLI 옵션: `--ken-burns-mode`

### 2. 자막 시스템

| 항목 | 현재 상태 | 문제점 |
|------|----------|--------|
| 스타일 | 배경 박스, 그림자 지원 | 해결됨 |
| AI 품질 | 품질 레벨 옵션 (creative/balanced/conservative) | 해결됨 |
| 줄바꿈 | 띄어쓰기 기반 분리 | 해결됨 |
| AI 자막 일치 | Phase 기반 자막 생성 | 해결됨 (Issue #22) |

**완료:** 자막 스타일 + 줄바꿈 + AI 품질 레벨 + Phase 기반 자막 생성

### 3. 음악/오디오

| 항목 | 현재 상태 | 문제점 |
|------|----------|--------|
| BGM | 단순 믹싱 | ~~동기화 없음~~ |
| 비트 매칭 | BPM 기반 동기화 | 해결됨 |
| 볼륨 | 고정 | 자동 조절 없음 |

**완료:** BGM 비트 동기화
- BPM 프리셋: slow (80), medium (110), upbeat (128), fast (150)
- 사진 전환 타이밍을 4비트 배수로 자동 정렬
- CLI 옵션: `--beat-sync <preset|bpm>`

### 4. 영상 구조

| 항목 | 현재 상태 | 문제점 |
|------|----------|--------|
| 인트로 | 4가지 프리셋 (simple, brand, minimal, cta) | 해결됨 |
| 아웃트로 | CTA 지원 (서브텍스트 포함) | 해결됨 |
| 로고 | 고정 위치 | 단순함 |

**완료:** 인트로/아웃트로 템플릿 시스템
- CLI 옵션: `--intro`, `--outro`, `--intro-preset`, `--outro-preset`
- 프리셋: simple, brand, minimal, cta

### 5. 썸네일

| 항목 | 현재 상태 | 문제점 |
|------|----------|--------|
| 선택 | 최적 프레임 자동 선택 (5개 후보) | 해결됨 |
| 텍스트 | 4가지 스타일 지원 | 해결됨 |
| 품질 | 파일 크기 기반 품질 분석 | 해결됨 |

**완료:** 썸네일 품질 개선
- 텍스트 오버레이: default, banner, centered, minimal
- 최적 프레임 선택: `--thumbnail-best` (파일 크기 기반)
- CLI 옵션: `--thumbnail-text`, `--thumbnail-style`

### 6. 이미지 정렬

| 항목 | 현재 상태 | 문제점 |
|------|----------|--------|
| 파일명 정렬 | 파일명 기반 정렬 | 해결됨 |
| AI 정렬 | Phase 분류 정확도 개선 (v3.2) | 해결됨 (Issue #21) |

**완료:** AI Phase 분류 정확도 개선
- before/after 구분 정확도 향상 (제외 키워드 도입)
- Phase 분류 프롬프트 대폭 개선 (판단 로직 명확화)
- Rate Limit 재시도 로직 추가 (안정성 강화)
